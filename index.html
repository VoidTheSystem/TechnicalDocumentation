Introduction

The SAP program allows us to manage business operations and customer relations. This documentation will cover the section that deals with the usage of SAP within the Technical Department.

The following sections will be covered:

  • Creating a notification for a product.

  • Creating an order that is linked to a notification.

  • Creating a quotation by using the order as a reference.

  • Accepting a quotation.

Notification Creation

The notification is a number that the Technical Department uses to log a problem. All notifications are problems that need to be solved by acting on them as soon as possible. When a customer books in a product, we create the problem notifaction for that product and we give the notification number to the customer in the form of a notification sheet.

On SAP, go to transaction IW51 (Create Service Notification). it will ask for a notifiation type, enter SN by typing it into the space or choose from the list by clicking the block next to the input space and press enter. on the next screen you only need to fill in the material which is the SAP code for the product you want to book in to SAP, for example D5EVO_R is the SAP code we use to identify a repair of the D5 Evo Sliding Gate Operator.

Order Creation

To create an order from a notification, you go to IW52 and insert the notification number for which you want to create an order. It will then open the page which allows you to change the notification. You will see two tabs on the top, one called service order and another called sales order. The service order will be chosen for most orders as this allows the order to sit on the system and wait to be accepted, unlike a sales order which needs to be processed once created. Once you selected service order, click on the Location tab and select fill in the three company details tabs. This step is very important as it tells the system which branch it needs to push the stock to, once the order has been accepted. Then select the Components tab and starting filling in the codes and quanitities required.

Take note that only physical stock items can be inserted in the components list.

Once all the components have been inserted, click on the Save icon at the top and the order number will be displayed in the infomation tab at the bottom of the screen, it is good practice to write this number on the notification sheet for easy access in the future.

Quotation Creation

To crate a quotation from the notification, the service order must first be completed as you cannot create a quotation without the order number which is used as a reference. If you do not have an order number, you would have to type out all the components on a new quotation which does not link to the notifaction.

Go to SAP operation DP80. This allows you to create a quotation using a reference, which is our order number in this case. If you have just created an order and not processed any other documents then you would notice that SAP has already inserterd your order number number. Enter the order number if it is not already entered, then press the Quotation Button at the top of the screen. The screen should change and you would now see a screen with all the items from the order. Some items will carry over as an item called materials and you need to check which item is missing from the list and replace materials with the correct item. The labour repair charge also needs to be inserted on the quotation depending on the item being repaired and zero cost needs to be applied to the applicable items, such as covers and o-rings. Once you are satisfied with the quotation, click the Save icon on the top left of the screen and the quotation will save and you will see the quotation number appear in the info bar at the bottom of the screen.
